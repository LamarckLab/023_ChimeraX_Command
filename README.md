## 📁 Overview

该仓库用于 **记录 ChimeraX 中常用的命令、快捷键、可视化技巧**，方便在结构生物学研究中快速查阅。

适用于：
- Cryo-EM 地图与结构拟合  
- PDB 操作、选择、标注  
- 模型导出、截取、着色  
- 结构比对、体素处理、残基选择等

| 文件 | 内容 |
